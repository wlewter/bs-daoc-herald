bs-daoc-herald
==============
Version 1.0.0

get the code with:
>git clone

change to the project directory
>cd bs-daoc-herald

install grunt-cli
>npm install -g grunt-cli

install bower
>npm install -g bower

install npm dependencies with:
>npm install

install bower dependencies with:
>bower install

run app locally with:
>grunt serve

build for herald distribution with:
>grunt build-herald
